How to create sample mock data?

create a sample javascript file and create a const with array and provide it module export, refer below example:

mock-data.js 
            const courses = [
                {
                id: 1,
                title: "Securing React Apps with Auth0",
                slug: "react-auth0-authentication-security",
                authorId: 1,
                category: "JavaScript"
                }
            ];

         const authors = [
            { id: 1, name: "Cory House" },
            { id: 2, name: "Scott Allen" },
            { id: 3, name: "Dan Wahlin" }
        ];

         const newCourse = {
            id: null,
            title: "",
            authorId: null,
            category: ""
        };

         // Using CommonJS style export so we can consume via Node (without using Babel-node)
        module.exports = {
            newCourse,
            courses,
            authors
        };

Run this command to initialize the mock data base / mock data json (i.e db.json)

        npm run init:mock-api-data

Run this command to start the mock-api

        npm run start:mock-api

run the below command to start the app in dev mode along with mock api after initializing the mock data base 

        npm run start:dev
